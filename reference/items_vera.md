# Small artificial item pool example.

A `data.frame` containing 80 items with different categorical and metric
properties.

## Usage

``` r
items_vera
```

## Format

A `data.frame` .

- item:

  Item identifier.

- exclusions:

  Items which can not be in the same test form.

- time:

  Average response times in minutes. `2.5` equals 2 minutes and 30
  seconds, for example.

- subitems:

  Number of sub items.

- MC, CMC, short_answer, open:

  Answer formats.

- diff_1, diff_2, diff_3, diff_4, diff5:

  Difficulty categories.
